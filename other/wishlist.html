wishlist

нестандартное вино

например из крыжовника или из фруктов типа личи/лонган/рамбутан

экзотические фрукты, ягоды и орехи

шоколад

  • без молока

  • не откровенно горький, то есть с сахаром/мёдом/etc. (но без химических подсластителей, такое можно жрать только по медицинским причинам)

  • без лецитина и прочей сои

(по моим наблюдениям, такое часто лежит рядом с извращениями типа косметики без глютена)

примеры:

  • Alce Nero

  • Amazing Cacao

  • Blanxart

  • Cluizel

  • HACHEZ

  • Nilambari

  • Stella

  • Vivani

  • Бритарев

  • Добро

  • Шоколад На Меду (похоже, испортился)

защитный переходник для USB, который пропускает только питание

не хочу чтобы зарядное устройство имело возможность притвориться клавиатурой (1, 2, 3) etc.

примеры:

  • Micro USB Data Block Charging Cable

  • 3rd Gen Data Blocker

  • SyncStop / USB Condom

фотокамера

  • карманный размер (грубо говоря, не больше Fuji X100)

  • RAW с поддержкой в dcraw/libraw

  • память без DRM, то есть CF или даже M.2

  • видоискатель с 100% покрытием кадра (в левом верхнем углу, иначе размер не карманный)

  • матрица

    • как минимум 4/3, а лучше APS

    • по возможности честная цветопередача (меня интересует красота перед камерой, а не “узнаваемый фирменный цвет”)

    • динамического диапазона много не бывает

    • форма чем ближе к квадрату, тем лучше (3:2 ещё нормально, 16:9 уже извращение)

  • объектив

    • ЭФР в районе 35-40, зум не подходит по размеру

    • светосилы много не бывает, особенно в питерских условиях

  • без неотключаемых Wi-Fi, Bluetooth etc.

  • без неотключаемых GPS/ГЛОНАСС/etc.

  • желательно без микрофона

  • ручные режимы (полностью ручной и приоритет выдержки/диафрагмы)

  • автоматический и ручной фокус (ручной фокус нужен редко, поэтому не требуется его максимальное удобство в качестве основного режима)

  • экран не поворотный, потому что он увеличивает толщину камеры (лучше потратить этот сантиметр на объектив)

  • без явного бреда типа неотключаемой вспышки (кстати, вспышка не нужна)

  • возможность сфокусироваться достаточно близко для документов (совсем уж насекомые не требуются)

  • питание от батареек, которые можно купить в деревенском магазине

  • без сенсорных экранов/кнопок (ну не нравятся они мне)

  • без неотключаемых посторонних звуков (при нажимании на кнопки etc.)

монитор

  • транспортабельный (на уровне “взять в отпуск/командировку и оставить в комнате”)

    • размер 15-20"

    • желательно складная конструкция с защитной крышкой (то есть в крайнем случае я могу сохранить коробку)

  • матовый

  • по возможности честная цветопередача

    • IPS или его аналоги (PLS, ADS)

    • желательно честный 8-битный цвет (на практике это означает псевдо-10-битный цвет)

    • желательно 100% покрытие sRGB (на практике это означает расширеный цветовой охват)

      • должна быть возможность включить режим sRGB кнопками на самом мониторе, чтобы софт мог не знать об этом

  • вертикальное разрешение как минимум 1200

  • форма 4:3, в крайнем случае 16:10 (это монитор, а не телевизор)

  • DisplayPort

    • желательно без HDMI (пусть засунут HDCP себе в телевизор)

  • без камеры и микрофона

  • желательно без динамиков

  • без неотключаемой самодеятельности (изменение яркости, резкости etc.)

  • без сенсорных экранов/кнопок (ну не нравятся они мне)

ищу названия моделей

сейчас они не нужны, но потом пригодятся

наручные часы

  • без логотипов и рекламных надписей (даже если бы я искал украшение, логотип в качестве украшения похож на мочу в качестве духов)

  • с батарейкой (и чтобы хватало на годы, поэтому смарты только с отдельной батарейкой для собственно часов)

  • прибор для измерения времени, а не бижутерия/дизайнерские/имиджевые/ювелирные/статусные

  • желательно с показом даты и дня недели

  • ремешок

    • кожа или ткань

    • не на липучке

    • стандартное крепление, чтобы не было проблем с заменой

  • не огромный размер

флэшка

  • механический переключатель защиты от записи

  • металлический корпус для защиты от электромагнитных воздействий (то есть в закрытом виде сплошная металлическая оболочка)

  • USB 3

  • как минимум 8 ГБ

  • без шифрования

Бывают отдельно защита от записи (1, 2, 3, 4) и металлический корпус (1, 2), а вместе не видел.

GPS-навигатор

на случай пеших походов

  • GPS (поддержка других систем приветствуется, но без GPS сразу нафиг)

  • питание от батареек, которые можно купить в деревенском магазине

  • быстрое позиционирование после включения (треки не нужны, хочу раз в несколько километров смотреть положение)

  • с возможностью использовать свои карты, например кусок OSM

  • память без DRM, то есть CF или даже M.2

  • желательно с возможностью просто посмотреть карту без GPS

  • желательно хоть как-то защищённый, чтобы не бояться уронить в лужу

  • без сенсорных экранов/кнопок (ну не нравятся они мне)

клавиатура

  • USB

  • отдельные блоки со стрелками и Ins/Del/Home/End/PgUp/PgDn

  • без цифрового блока

  • трекпойнт (и никаких тачпадов, мне не нужна клавиатура размером с ноутбук)

  • без кнопки выключения над стрелками (её нужно установить на лоб тому, кто это придумал)

  • чем тише, тем лучше

  • без неотключаемой подсветки

  • без извращений типа логотипа на пробеле

  • с русскими буквами

  • шрифты нормальные, а не игровые

(Space Saver II давно не производят)

телефон

такое впечатление, что телефоны специально портят, чтобы вместо них покупали смартфоны

  • не Alcatel (они не работают со старыми мегафоновскими SIM-картами)

  • телефон для звонков, а не недопланшет

  • простая прочная форма, никаких раскладушек/слайдеров

  • обычный внешний вид, не “для пенсионеров”

  • желательно монохромный экран

  • громкий разборчивый динамик (для сравнения: в Nokia 1100 хороший, в Nokia 1280 говно)

  • аккумулятор

    • большой, как минимум 2-3 недели ожидания (не понимаю, зачем добавляют телефонные функции в устройства, которым заряда хватает меньше чем на неделю)

    • заменяемый

    • желательно совместимый с чем-то ещё

  • без GPS/ГЛОНАСС/etc. (не хочу облегчать слежку сверх необходимого для связи минимума)

  • без сканеров отпечатков пальцев, радужки глаза etc.

  • без камеры

  • без неотключаемых Wi-Fi, Bluetooth etc.

  • без акселерометра, гироскопа, компаса, барометра и прочих лишних сенсоров (меня тоже удивляет, что такой вопрос в принципе возникает)

  • при написании СМС и контактов должны получаться мои буквы без “исправлений”

  • не умеющий быть привязаным к оператору, чтобы левая SIM-карта не могла привязать

  • интерфейс написан людьми, которые видят телефон не первый раз в жизни (пробовал Philips Xenium X2300, он сделан папуасами для папуасов)

  • желательно с аппаратным выключателем микрофона, чтобы не слушал лишнего

  • без сенсорных экранов/кнопок (ну не нравятся они мне)

  • без неотключаемых посторонних звуков (при нажимании на кнопки etc.)

внешняя звуковая карта

  • аппаратная поддержка 44,1 кГц

  • честный/мониторный звук без “улучшений” (если музыкальный жанр важен, то эта модель не подходит)

  • просто ЦАП, а не кирпич с десятками регуляторов

  • какое-то подобие усилителя, чтобы хватило мощности для студийных наушников (усилитель в “тёплом ламповом” смысле не нужен)

  • питание от USB

(из годных моделей пробовал NuForce uDAC-3)

наушники

  • закрытые

  • с проводом

  • честный/мониторный звук без “улучшений” (если музыкальный жанр важен, то эта модель не подходит)

(из годных моделей пробовал Beyerdynamic DT 150 и Roland RH-300)

папка для хранения разнообразных бумажек (документы/инструкции/чеки/справки/etc.)

  • побольше небольших отделений (карманы для кредиток/визиток не нужны, такой мелочи у меня почти нет)

  • по возможности без логотипов, особенно блестящих и орнамента из логотипов

  • материал не совсем уж говно типа хрупкого пластика

Кстати, достаточно размера около А5 или тетради. Найти отдельную папку только под А4 совершенно не проблема, этого добра как грязи.

(в качестве временного решения использую сумку для ноутбука, но это не эталон компактности)
